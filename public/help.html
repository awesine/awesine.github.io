it doesnt unblock things that require an active connection (like twitch, youtube, etc)
it just saves a snapshot of the site

to unblock reddit change https://reddit.com/[whatever] to https://old.reddit.com/[whatever] (if you click on a reddit link and it breaks/uses the new UI it probably went to a reddit.com instead of an old.reddit.com, just add the "old." in the URL bar)
if you want to go to a specific subreddit input old.reddit.com/r/[subreddit] it seems that reddit is broken and just gives you a server error. rip. use archive.is instead

flash games work (most at least, i checked maxgames and it worked)

tumblr works if you go to a specific blog (some blog themes might mess up the unblocker, in that case use one of the other sites below)

you cant log in onto anything so you cant use discord or scroll your personalized reddit/tumblr feed or whatever

if a site doesn't work or starts to not work or breaks for some reason, try to save it using http://archive.is/
if archive.is doesn't work, use http://web.archive.org/
if web.archive.org doesn't work then you're out of luck
